⚙️ 6. Production-Grade Technical Logic
Backend Aggregation (The "Brain")
To handle 80,000 rows instantly, do not run raw queries. Use a PostgreSQL Materialized View:

SQL
CREATE MATERIALIZED VIEW leaderboard_stats AS
SELECT 
    dept_name,
    COUNT(event_id) as total_events,
    SUM(registrations) as total_regs,
    RANK() OVER (ORDER BY SUM(registrations) DESC) as position
FROM event_summary
GROUP BY dept_name;
